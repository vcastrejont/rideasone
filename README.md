#### Back end stack:

 - Nodejs
 - MongoDB
 - Express
 - Mongoose
 


*Installing Back end components:*
```sh
$ npm  install
```

#### Front end stack:

 - Angular
 - Bootstrap
 - Angular maps
 - Less
 
*Installing Front end components:*
```sh
$ bower install
```

#### Run carpooling app

*execute the next command to run the app*
```sh
$ npm start
```

*Open your browser at:*
```sh
http://localhost:3000/
```
