# Nearsoft Car Pooling app

Corporate web system to share your travel by car to and from work


#### Back end stack:

 - Nodejs
 - MongoDB
 - Express
 - Mongoose

*Installing Back end components:*
```sh
$ npm  install
```

#### Front end stack:

 - Angular
 - Bootstrap
 - Angular maps
 - Less
 
*Installing Front end components:*
```sh
$ grunt install
```
